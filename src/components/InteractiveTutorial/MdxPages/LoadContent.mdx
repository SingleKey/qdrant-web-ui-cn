# 从远程快照加载数据到集合

在本教程中，我们将指导您如何从远程快照将数据加载到Qdrant集合中。

## 步骤1：将快照导入到集合

首先，创建集合`midjourney`并将向量数据加载到其中。
该集合将采用快照的参数，向量大小为512，相似度使用余弦距离进行测量。

```json withRunButton=true
PUT /collections/midjourney/snapshots/recover
{
  "location": "http://snapshots.qdrant.io/midlib.snapshot"
}
```

等待片刻，快照中的向量将被添加到`midjourney`集合中。

## 步骤2：验证数据上传

数据导入后，验证其是否成功上传非常重要。您可以通过检查集合中的向量（或点）数量来实现这一点。

运行以下请求以获取向量计数：

```json withRunButton=true
POST /collections/midjourney/points/count
```

该集合应包含5,417个数据点。

### 步骤3：打开集合UI
您还可以[检查您的集合](/dashboard#/collections/midjourney/)以查看上传的数据。
